# Behind The Shore

2D екшн-пригодницька гра, створена на Unity з динамічним ШІ ворогів, бойовою механікою та геймплеєм дослідження.

[![Unity Version](https://img.shields.io/badge/Unity-2022.3+-blue.svg)](https://unity.com/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## Про гру

Behind The Shore - це насичена екшеном 2D гра, де гравці повинні збирати містичні руни, борючись з ворогами у таємничому світі. Особливості:

- **Динамічний ШІ ворогів** - Вороги з поведінкою блукання, переслідування та атаки
- **Бойова система** - Стратегічна бойова механіка
- **Управління здоров'ям** - Шкала здоров'я з системою регенерації
- **Предмети для збору** - Знаходьте та збирайте руни, щоб перемогти
- **NavMesh пошук шляху** - Розумна навігація ворогів за допомогою Unity NavMesh

## Як грати

- **WASD** або **Стрілки** - Рух персонажа
- **Миша** - Напрямок прицілювання
- **ESC** - Меню паузи
- **Мета** - Зібрати всі руни, щоб перемогти!

## Завантаження та встановлення

### Варіант 1: Грати у скомпільовану гру (Рекомендовано)

1. Перейдіть до [Releases](../../releases)
2. Завантажте останній `BehindTheShore_Build.zip`
3. Розпакуйте архів
4. Запустіть `BehindTheShore.exe` (Windows) або відповідний виконуваний файл для вашої платформи
5. Насолоджуйтесь грою!

### Варіант 2: Відкрити в Unity Editor

**Вимоги:**
- Unity 2022.3 або новіша версія
- Git встановлений у вашій системі

**Кроки:**

1. **Клонуйте репозиторій:**
   ```bash
   git clone https://github.com/yourusername/behind-the-shore.git
   cd behind-the-shore
   ```

2. **Відкрийте в Unity:**
   - Запустіть Unity Hub
   - Натисніть "Open" або "Add"
   - Перейдіть до папки клонованого репозиторію
   - Виберіть папку проєкту та натисніть "Open"

3. **Грайте в редакторі:**
   - Дочекайтеся імпорту всіх ресурсів Unity
   - Відкрийте сцену `MainMenu` з `Assets/Scenes/`
   - Натисніть кнопку Play в Unity Editor

## Технічні деталі

### Створено за допомогою
- **Рушій:** Unity 2022.3+
- **Мова:** C#
- **ШІ навігація:** Unity NavMesh Plus
- **Система вводу:** Нова система вводу Unity

### Ключові системи

**ШІ ворога (EnemyAI.cs)**
- Машина станів зі станами Idle, Roaming, Chasing та Attacking
- Пошук шляху на основі NavMesh
- Налаштовувані діапазони виявлення та атаки
- Динамічне регулювання швидкості

**Система гравця (Player.cs)**
- Управління здоров'ям з регенерацією
- Механіка відновлення після пошкоджень
- Фізика відкидання
- Обробка вводу через систему вводу Unity

**Бойова система**
- Виявлення попадань за допомогою колайдерів
- Розрахунок пошкоджень
- Візуальний зворотній зв'язок з ефектами спалаху
- Відкидання при пошкодженні
